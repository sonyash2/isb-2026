# Лабораторная работа №4
## Контроль целостности файлов с использованием хеш-функций

### Цель работы
Изучить принципы работы криптографических хеш-функций и реализовать механизм проверки целостности файлов.

### Функциональность

1. **Вычисление хеша файла** (SHA-256, MD5, SHA-1)
2. **Сохранение контрольной суммы** в отдельный файл
3. **Проверка целостности** файла
4. **Визуализация прогресса** (tqdm в CLI, QProgressBar в GUI)
5. **Поиск коллизий** (демонстрация)
